# AI Coding CLI Tools 学习项目

本项目用于学习、总结和梳理不同的开源 AI 编程 Agent 工具。

## 包含的子项目

### 1. [OpenAI Codex](./codex)
OpenAI 的代码生成 CLI 工具。
- 仓库: https://github.com/openai/codex

### 2. [Google Gemini CLI](./gemini-cli)
Google 的 Gemini 命令行工具。
- 仓库: https://github.com/google-gemini/gemini-cli

### 3. [OpenCode](./opencode)
开源的 AI 编程助手工具。
- 仓库: https://github.com/anomalyco/opencode

## 学习目标

- 理解不同 AI 编程 Agent 的架构设计
- 比较各工具的实现方式和特点
- 总结最佳实践和设计模式

## 使用说明

克隆本项目及子模块：
```bash
git clone --recursive git@github.com:hongyesuifeng/AICodingCLI.git
```

或已克隆后拉取子模块：
```bash
git submodule update --init --recursive
```
